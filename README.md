# Physics_Avatar_Follower  
A system for a physics / rigidbody based synced VRChat avatar follower.  
> [!IMPORTANT]
> This system uses some techniques and parts from [VRLab's rigidbody throw system](https://github.com/VRLabs/Rigidbody-Throw)  
> Major credits to them and everything they make, it's awesome, please go check them out.
 
<img width="800" height="423" alt="PAF_Preview" src="https://github.com/user-attachments/assets/f6ad09bf-2bfe-448e-9d56-365faf93bab1" />
 
This system included the prefab, FX controller, parameters list, and menu.  
 
It originally spawns at (0,0,0) when in-game, you may need to toggle it off and back on to respawn it if it goes missing.  

You can just drag and drop any model you want underneath of the "Follower" game object, where the car model is.  

Car model and textures are just for looks and made by me, they can be found [here.](https://sketchfab.com/3d-models/stylized-low-poly-cars-a46f905e967c40c99c637fcfb5a2ba75)  

This system was originally created for my [deep sea bunny avatar follower asset.](https://sfdvrc.gumroad.com/l/Bunny)  

<img width="276" height="576" alt="image" src="https://github.com/user-attachments/assets/9f7af2a9-f7c4-4f7c-839e-318941f0042b" />

### Uses:  
- 18 Constraints
- 2 Colliders
- 2 Rigidbody's
- 6 Phys Bones
- 6 Phys Bone Transforms
- 3 Phys Bone Colliders
- 1 Particle System
- 1 Max Particle Count
- 1 VRC Raycast
